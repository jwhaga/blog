# Aurora 前端组件与路由结构

> 本文档描述前台（aurora-blog）和后台（aurora-admin）的组件层次、路由配置和 Store 划分。

---

## 一、前台（aurora-blog）

### 1.1 路由表

| 路径 | 视图组件 | 说明 |
|------|---------|------|
| `/` | Home.vue | 首页（文章列表 + 侧边栏） |
| `/articles/:articleId` | Article.vue | 文章详情 |
| `/talks` | TalkList.vue | 说说列表 |
| `/talks/:talkId` | Talk.vue | 说说详情 |
| `/archives` | Archives.vue | 文章归档（时间线） |
| `/article-list/:tagId` | ArticleList.vue | 按标签筛选文章列表 |
| `/tags` | Tags.vue | 标签云页面 |
| `/about` | About.vue | 关于我 |
| `/message` | Message.vue | 留言板 |
| `/friends` | FriendLink.vue | 友情链接 |
| `/photos/:albumId` | Photos.vue | 相册浏览 |
| `/oauth/login/qq` | OauthLogin.vue | QQ 登录回调 |
| `/404` | 404.vue | 404 页面 |
| `/:catchAll(.*)` | → 重定向到 /404 | 未匹配路由 |

### 1.2 组件层次结构

```
App.vue
├── AuroraNavigator.vue       # 顶部导航栏
├── Header/                   # 页面头部组件
│   ├── Feature.vue           # 首页特色/横幅区域
│   ├── Avatar.vue            # 用户头像组件
│   ├── Breadcrumb.vue        # 面包屑导航
│   └── MobileMenu.vue        # 移动端菜单
├── Sidebar/                  # 侧边栏组件
│   ├── LoadingSkeleton/      # 骨架屏加载状态
│   ├── Tag/                  # 标签展示
│   ├── Title/                # 标题组件
│   ├── Dropdown/             # 下拉菜单
│   └── ToggleSwitch/         # 开关组件
├── <router-view>             # 页面内容区域
│   ├── Home.vue              # 首页
│   ├── Article.vue           # 文章详情
│   ├── Talk.vue / TalkList.vue
│   └── ...其他页面视图
├── Comment/                  # 评论组件
├── ArticleCard/              # 文章卡片
├── Paginator.vue             # 分页组件
├── SearchModel.vue           # 搜索弹窗
├── Sticky.vue                # 粘性定位组件
├── Footer.vue                # 页脚
├── Social.vue                # 社交链接
├── UserCenter.vue            # 用户中心弹窗
├── Dia.vue                   # 模态对话框
└── OauthLogin.vue            # QQ 登录回调
```

### 1.3 Store（Pinia）划分

| Store 文件 | 说明 |
|-----------|------|
| app.ts | 应用全局状态（主题、语言等） |
| article.ts | 文章列表、当前文章、归档数据 |
| Category.ts | 分类列表 |
| comment.ts | 评论数据 |
| common.ts | 通用状态 |
| dia.ts | 对话框状态 |
| dropdown.ts | 下拉菜单状态 |
| local.ts | 本地存储持久化 |
| meta.ts | 页面元信息（标题、描述等） |
| navigator.ts | 导航栏状态 |
| photo.ts | 相册/照片状态 |
| search.ts | 搜索状态 |
| tag.ts | 标签状态 |
| user.ts | 用户登录状态 |

### 1.4 主要依赖

| 依赖 | 用途 |
|------|------|
| Vue 3 + TypeScript | 基础框架 |
| Element Plus | UI 组件库 |
| Pinia | 状态管理 |
| Vue Router 4 | 路由 |
| Axios | HTTP 请求 |
| markdown-it + PrismJS | Markdown 渲染 + 代码高亮 |
| mavon-editor | Markdown 编辑器 |
| Tailwind CSS | 原子化 CSS |

## 三、前后台路由差异说明

| 特性 | 前台（aurora-blog） | 后台（aurora-admin） |
|------|--------------------|---------------------|
| Vue 版本 | 3.x | 2.6.14 |
| TypeScript | 是（~4.5.5） | 否（纯 JS） |
| 状态管理 | Pinia | Vuex 3 |
| Router | Vue Router 4 | Vue Router 3 |
| UI 库 | Element Plus | Element UI |
| 构建工具 | @vue/cli ~5.0.0 | @vue/cli ~5.0.0 |
| 代码风格 | `<script setup>` | Vue 2 Options API |

> **重构方向**：Phase 3 计划将后台迁移到 Vue 3 + Element Plus + Pinia，与前台统一技术栈。
