# 配置说明文档

本文档说明本项目中的所有可配置外部服务器地址及修改方式。

---

## 一、前端配置（Vue CLI 环境变量）

前端项目使用 `.env.development`（开发环境）和 `.env.production`（生产环境）文件来管理外部服务器地址。修改对应文件中的值即可切换环境，无需修改代码。

### 1.1 aurora-admin（管理后台）

**文件位置：** `aurora-vue/aurora-admin/.env.development` / `.env.production`

| 变量名 | 说明 | 开发环境示例 | 生产环境示例 |
|--------|------|------------|------------|
| `VUE_APP_OSS_URL` | 对象存储基础 URL，拼接图片/静态资源路径 | `http://localhost:9000/aurora` | `https://static.yourdomain.com/aurora` |
| `VUE_APP_BLOG_URL` | 博客网站域名 | `http://localhost:8080` | `https://www.yourdomain.com` |

### 1.2 aurora-blog（博客前端）

**文件位置：** `aurora-vue/aurora-blog/.env.development` / `.env.production`

| 变量名 | 说明 | 开发环境示例 | 生产环境示例 |
|--------|------|------------|------------|
| `VUE_APP_OSS_URL` | 对象存储基础 URL | `http://localhost:9000/aurora` | `https://static.yourdomain.com/aurora` |
| `VUE_APP_BLOG_URL` | 博客网站域名 | `http://localhost:8080` | `https://www.yourdomain.com` |
| `VUE_APP_QQ_APP_ID` | QQ 互联 APP ID | 留空 | `101999415` |
| `VUE_APP_QQ_REDIRECT_URI` | QQ 登录回调地址 | `http://localhost:8080/oauth/login/qq` | `https://www.yourdomain.com/oauth/login/qq` |
| `VUE_APP_GONGAN_ICON_URL` | 公安备案图标 URL | `http://localhost:9000/aurora/config/gongan-beian-icon.png` | `https://static.yourdomain.com/aurora/config/gongan-beian-icon.png` |

## 二、后端配置（Spring Boot YAML）

### 2.1 开发环境

**文件位置：** `aurora-springboot/src/main/resources/application-dev.yml`

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `server.port` | 后端服务端口 | `8080` |
| `upload.url` | 对象存储访问基础 URL | `http://localhost:9000/aurora` |
| `upload.mode` | 上传模式（oss/minio） | `minio` |
| `upload.minio.url` | MinIO 服务地址 | `http://localhost:9000` |
| `website.url` | 网站部署地址 | `http://localhost:8080` |
| `qq.app-id` | QQ 互联 APP ID | 空 |

其他基础设施配置：MySQL (`localhost:3306`)、Redis (`localhost:6379`)、RabbitMQ (`localhost:5672`)、ElasticSearch (`localhost:9201`)

### 2.2 生产环境

**文件位置：** `aurora-springboot/src/main/resources/application-prod.yml`

所有配置项通过环境变量注入，部署前需设置：

| 环境变量 | 说明 | 示例 |
|----------|------|------|
| `OSS_URL` | 对象存储访问域名 | `https://static.yourdomain.com/aurora` |
| `WEBSITE_URL` | 网站域名 | `https://www.yourdomain.com` |
| `QQ_APP_ID` | QQ 互联 APP ID | `101999415` |
| `MYSQL_URL` | 数据库连接地址 | `jdbc:mysql://...` |
| `REDIS_HOST` | Redis 地址 | `...` |
| `ES_URIS` | ElasticSearch 地址 | `http://...` |

## 三、部署流程

### 第一步：配置基础设施地址

```bash
# 启动本地基础设施（开发环境）
# MySQL、Redis、RabbitMQ、ElasticSearch、MinIO
```

### 第二步：修改配置文件

```
# 前端 - 修改 .env.production
VUE_APP_OSS_URL=https://static.yourdomain.com/aurora
VUE_APP_BLOG_URL=https://www.yourdomain.com

# 后端 - 修改 application-prod.yml 或设置环境变量
OSS_URL=https://static.yourdomain.com/aurora
WEBSITE_URL=https://www.yourdomain.com
```

### 第三步：构建前端

```bash
cd aurora-vue/aurora-admin
npm install
npm run build          # 构建产物在 dist/ 目录

cd ../aurora-blog
npm install
npm run build          # 构建产物在 dist/ 目录
```

### 第四步：部署后端

```bash
cd aurora-springboot
mvn clean package      # 构建 jar
java -jar target/aurora-*.jar
```
