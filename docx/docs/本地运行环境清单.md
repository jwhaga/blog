## 二、中间件（Docker 容器）

所有中间件通过 Docker Compose 一键启动，详见根目录下的 [docker-compose.yml](docker-compose.yml)。

| 服务 | 容器名 | 端口 | 用户名 | 密码 |
|------|--------|------|--------|------|
| MySQL 8.0 | `aurora-mysql` | 3306 | `aurora` | `aurora123` |
| Redis 7.2 | `aurora-redis` | 6379 | — | `aurora123`（requirepass） |
| RabbitMQ 3.13 | `aurora-rabbitmq` | 5672 / 15672 | `aurora` | `aurora123` |
| Elasticsearch 8.15.3 + IK | `aurora-es` | 9201→9200 | — | —（xpack 已关闭） |

> 注意：本机 9200 端口可能被其他服务占用（如 Cpolar），ES 容器内映射到 9201。

## 三、本地开发配置

### 3.1 Docker 启动顺序

```powershell
# 启动所有基础设施服务
docker compose up -d

# 确认服务就绪（healthcheck 通过后 STATUS 显示 healthy）
docker ps
```

### 3.2 编译与测试

```powershell
# 设置 JDK 25 环境
$env:JAVA_HOME="D:\Develop\JDK25\jdk-25.0.3+9"

# 编译
d:\Develop\Maven\apache-maven-3.9.16\bin\mvn.cmd clean compile -f aurora-springboot\pom.xml -q

# 运行所有单元测试
d:\Develop\Maven\apache-maven-3.9.16\bin\mvn.cmd clean test -f aurora-springboot\pom.xml
```

### 3.3 启动后端

```powershell
# 打包
d:\Develop\Maven\apache-maven-3.9.16\bin\mvn.cmd clean package -f aurora-springboot\pom.xml -DskipTests -q

# 启动（Dev 模式，默认配置）
D:\Develop\JDK25\jdk-25.0.3+9\bin\java.exe -jar aurora-springboot\target\aurora-springboot-0.0.1.jar --spring.profiles.active=dev --server.port=8081
```

### 3.4 启动前端

```powershell
# 前台（Vue 3）
cd aurora-vue/aurora-blog
npm install
npm run dev

# 后台（Vue 2）
cd aurora-vue/aurora-admin
npm install
npm run serve
```

### 3.5 本地覆盖配置

如需覆盖本地开发配置（如修改密码、端口），可创建 `application-dev-local.yml`：

```powershell
cp aurora-springboot/src/main/resources/application-dev-local.yml.example aurora-springboot/src/main/resources/application-dev-local.yml
```

编辑该文件后重启后端即可生效，该文件已被 `.gitignore` 排除，不会被提交到版本库。

## 四、验证接口

后端启动后可通过以下接口验证：

```powershell
# 核心接口冒烟测试
$tests = @("/", "/articles/all", "/categories/all", "/tags/all", "/about")
foreach ($url in $tests) {
    Invoke-RestMethod -Uri "http://localhost:8081$url" -TimeoutSec 10
}

# 登录获取 Token
$body = @{ username = "admin@163.com"; password = "admin" }
Invoke-RestMethod -Uri "http://localhost:8081/users/login" -Method POST `
    -ContentType "application/x-www-form-urlencoded" -Body $body

# Knife4j API 文档
Invoke-WebRequest -Uri "http://localhost:8081/doc.html" -UseBasicParsing
```

## 五、常用命令速查

| 操作 | 命令 |
|------|------|
| 编译后端 | `mvn clean compile -f aurora-springboot\pom.xml` |

| 停止 Docker 服务 | `docker compose down` |
| 验证脚本 | `.\verify.ps1` |
