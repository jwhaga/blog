# Aurora 博客系统 — 代码功能目录

## 根目录 `重构版/`

| 文件/目录 | 功能说明 |
|-----------|----------|
| `aurora-springboot/` | 后端项目（Java Spring Boot + Maven） |
| `aurora-vue/` | 前端项目（Vue 2 + Vue 3 双前端） |
| `docx/` | 清理出的无用文档/日志存档 |
| `.gitattributes` | Git 属性配置（换行符处理等） |
| `.gitignore` | Git 忽略规则 |
| `docker-compose.yml` | Docker 编排文件（一键启动 MySQL/Redis/MinIO/RabbitMQ） |
| `Dockerfile.es` | Elasticsearch 带 IK 分词插件的构建配置 |
