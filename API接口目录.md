| ResultVO<T>: { code: number, message: string, data: T } |\n| 认证方式 | Authorization: Bearer <jwt> |\n| 成功码 | 200 |\n| 失败码 | 400（参数错误）、401（未登录）、403（无权限）、500（服务异常） |\n\n---\n\n## 二、文章模块\n\n| 方法 | 路径 |
